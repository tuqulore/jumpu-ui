# Label

Jumpu UI では **label 要素に対してスタイルは提供していません**、その理由は以下の通りです。

## 状況によってスタイルが異なるため

状況によっていくつかのスタイルを使い分ける必要があります。例えば名前のフィールドと複数のチェックボックスだとレイアウトが異なることがしばしば起こります。

## デザイナーの意図を柔軟に反映させたいため

Jumpu UI のコンセプトとして、デザイナーの意図が柔軟に反映できるようスタイルを付与する設計となっています。そのため label 要素を強制的にスタイルで縛るのはコンセプトに合致しません。

## 他のフレームワークとの差別化

開発者がこのフレームワークを選ぶか選ばないかという判断がしやすくなります。
