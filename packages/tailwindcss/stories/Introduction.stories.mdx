# @jumpu-ui/tailwindcss

@jumpu-ui/tailwindcss パッケージは Jumpu UI の TailwindCSS プラグインです。

## Figma

デザインツール Figma でも Jumpu UI が公開されています。

[Jumpu UI （Figma 版）](https://www.figma.com/community/file/1042706969451783914/%5Bdev%5D-Jumpu-UI-%7C-Tailwind-CSS-Components-%26-Design-System)

## 使い方

コンポーネントごとにストーリーが用意されています。

使いたいコンポーネントの見た目がある時は、次の手順でマークアップを確認します。

1. Docs タブに切り替える
2. Show code ボタンをクリックする
