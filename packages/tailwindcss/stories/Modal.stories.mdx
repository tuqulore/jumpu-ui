# Modal

Jumpu UI ではモーダルに`dialog`要素を使っています。`dialog`要素は非常にシンプルな機能をネイティブで提供してくれます。
原則は JavaScript と併用で使い、`showModal()` メソッドと `close()` メソッドだけで開閉するだけでなく、背景色の指定は CSS`::backdrop`疑似要素に指定すればよく、開いている状態でエスケープキーを押すと閉じるような機能も提供します。

`open` 属性は最初は外しておき、JavaScript で呼び出すようにしてください。
`::backdrop`疑似要素は`showModal()`の呼び出し時に表示されます。

```
const dialog = document.querySelector(".jumpu-dialog");
dialog.showModal()

```

参考 https://developer.mozilla.org/en-US/docs/Web/API/HTMLDialogElement/showModal
