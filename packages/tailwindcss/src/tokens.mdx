# Tokens

## Colors

ステータス通知に用いる色を定義しています。コントラストの関係上、原則白抜きもしくは黒抜きで用いなければなりません。

- <span class="inline-flex rounded-full bg-success text-white text-sm px-3 py-0.5">#15803D</span> 成功：ユーザー対して、操作が正常に完了した時に使用します。
- <span class="inline-flex rounded-full bg-warning text-black text-sm px-3 py-0.5">#FACC15</span> 警告：ユーザーに対して、これからおこなう操作に注意を払う必要がある時に使用します。
- <span class="inline-flex rounded-full bg-danger text-white text-sm px-3 py-0.5">#B80000</span> 危険：ユーザーに対して、取り消せない操作など最大限の注意を払う必要がある時に使用します。
