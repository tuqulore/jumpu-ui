## 準備

パッケージマネージャーとして [pnpm](https://pnpm.io/) を採用しています。ご自身の開発環境に適切な方法で[インストール](https://pnpm.io/ja/installation)してください。

## クイックスタート

```shell
# パッケージのインストール
pnpm install
# コードの検査
pnpm -r lint
# 開発サーバーの起動
pnpm -r dev
```
